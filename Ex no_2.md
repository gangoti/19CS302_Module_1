# EX 2 C program to check whether the attendance is PRESENT using simple if statement.
## DATE:
## AIM:
To write a program to check whether the attendance is PRESENT using simple if statement.

## Algorithm
Start. Declare a variable value of type char. Prompt the user to enter a value. Read the value using scanf. Check whether the attendance is PRESENT. If true, print "Present". If false, print " ". End.

## Program:
```
#include<stdio.h> 
int main()
{
char p1; 
scanf("%c", &p1);
if(p1=='P')
{
printf("Present");
}
return 0;
}
```
